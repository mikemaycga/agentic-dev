# Knowledge Library

This library contains curated external knowledge that has been evaluated through the Contrarian Analyst process.

## Structure

| Directory | Contents |
|-----------|----------|
| `articles/` | Saved articles, blogs, transcripts — raw source material |
| `videos/` | Video references with links and notes |
| `assessments/` | Contrarian Analyst evaluations of external sources |
| `adopted/` | Content that directly influenced framework standards or decisions |

## Process

1. External content is submitted for evaluation
2. Contrarian Analyst produces an assessment in `assessments/`
3. Raw source material is archived in `articles/` or `videos/`
4. Human reviews the assessment and makes adopt/consider/archive/reject decisions
5. Adopted content is cross-referenced in `adopted/` with links to the standards it influenced

## Naming Convention

- Assessments: `YYYY-MM-DD-source-name-topic.md`
- Articles: `YYYY-MM-DD-source-name-topic.md` or `YYYY-MM-DD-source-name-topic-transcript.txt`
- Videos: `YYYY-MM-DD-source-name-topic.md` (notes with video link)
